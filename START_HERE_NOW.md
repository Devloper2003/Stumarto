## 🔧 Optional Upgrades

### Add Real Database (Optional)
```
1. Create MongoDB cluster at mongodb.com/cloud/atlas
2. Get connection string
3. Add MONGODB_URI variable to Railway
4. Restart service
```

### Enable Live Payments (Optional)
```
1. Get Razorpay live keys at razorpay.com
2. Update RAZORPAY_KEY_* variables in Railway
3. Set PAYMENT_MODE=live
4. Restart service
```

### Add Custom Domain (Optional)
```
1. Railway dashboard → Your project
2. Settings → Custom domain
3. Update DNS records
```

## 🆘 Quick Troubleshooting

### App won't load?
- Check internet connection
- Hard refresh (Ctrl+Shift+R)
- Clear browser cache
- Check Railway logs

### API not working?
- Check Railway logs for errors
- Verify endpoint URLs (/api/products, /api/cart, etc)
- Check backend .env variables

### Payment failing?
- App is in test mode (normal)
- Test mode cards work fine for testing
- To enable live payments: add real Razorpay keys

### Need help?
- Check [DEPLOYMENT_FINAL.md](DEPLOYMENT_FINAL.md) troubleshooting
- View Railway logs (dashboard → logs)
